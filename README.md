# Marching Soldiers Game

A simple Three.js game featuring soldiers marching forward on a hovering platform.

## How to Play

1. Use the LEFT and RIGHT arrow keys to move the soldiers
2. Collect green bonuses to add more soldiers to your squad
3. Avoid red penalties that will reduce your squad size
4. Don't fall off the sides of the platform!
5. Try to reach the end of the platform with as many soldiers as possible

## Game Features

- Soldiers automatically march forward on a hovering platform
- Green bonus items increase your squad size
- Red penalty items decrease your squad size
- Game over if all soldiers are lost or if they fall off the platform
- Simple controls: left/right movement only

## Running the Game

Just open `index.html` in your web browser. No server required!

For the best experience, use a modern browser with WebGL support like Chrome, Firefox, or Edge.

## Development

This game uses:
- Three.js for 3D rendering
- Pure JavaScript (no additional frameworks)
- HTML5 and CSS3
